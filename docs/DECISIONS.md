| D-031 | Publish the recorded standalone dashboard as the no-install judge path.                                                                                   | Human + Codex implementation | Developer-tool submissions require a test path without rebuilding; the artifact is useful if clearly labeled as recorded.                                                                               | A safe live sandbox replaces the recorded artifact.                                                                                   |

| D-033 | Commit the bundled Node 24 Action entry point while keeping the CLI as a release artifact.                                                                | Codex implementation         | GitHub Actions execute repository contents directly; an ignored Action bundle makes every source ref uninstallable.                                                                                     | GitHub supports building JavaScript actions before execution.                                                                         |

| D-035 | Freeze a deadline-locked must-ship scope while preserving the full product roadmap separately.                                                            | Human                        | The complete roadmap is a multi-week program; finishing a smaller evidence-first scope protects the working product and deadline.                                                                       | The July 21, 2026 product cutoff has passed and the deadline build is frozen.                                                         |
| D-036 | Disable or defer an under-proven feature rather than weaken evidence, isolation, verification, or required checks.                                        | Human + Codex implementation | A deadline cannot justify false confidence or a weaker security contract.                                                                                                                               | None.                                                                                                                                 |
| D-037 | Require one immutable witness digest, structured outcomes, legitimate success, and exact architecture evidence for every new `verified` transition.       | Human + Codex implementation | A generic nonzero exit, changed witness, or unrelated graph delta is not counterfactual proof.                                                                                                          | None.                                                                                                                                 |
| D-038 | Validate Codex patches in a secretless, read-only job before a separate publisher may open one draft PR.                                                  | Human + Codex implementation | Model output must cross bounded path, mode, size, digest, stale-head, and target-test gates before receiving write authority.                                                                           | A first-party patch broker replaces the workflow handoff.                                                                             |
