# OpenAI Build Week challenge requirements

This file preserves the challenge brief supplied during planning. Reverify the official rules before submission.

## Build requirement

Create a working project using Codex with GPT-5.6. Valid formats include an app, agent, website, game, workflow, developer tool, or another original project.

## Tracks

- Apps for your life
- Work and productivity
- Developer tools
- Education

Hedge targets **Developer tools**.

## Required submission elements

- A working project.
- A selected category.
- A project description explaining what was created and how it works.
- A public YouTube demonstration under three minutes.
- Audio explaining how both Codex and GPT-5.6 were used.
- A repository URL for judging and testing.
- Public repository with relevant licensing, or private access shared with the two judging addresses listed in the official brief.
- README with setup instructions, sample data if required, and clear run guidance.
- Clear explanation of where Codex accelerated development, which key decisions were made, and how GPT-5.6 and Codex were used.
- The `/feedback` Codex Session ID from the session where the majority of core functionality was built.

## Timing supplied in the brief

- Request free Codex credits by Friday, July 17, 2026 at 12:00 PM Pacific.
- Submit by Tuesday, July 21, 2026 at 5:00 PM Pacific.
- Internal Hedge target: noon Pacific on July 21.

## Evidence discipline

- Preserve dated commits.
- Mark any pre-existing baseline.
- Keep the main implementation in the primary Codex session.
- Record human versus model decisions.
- Do not overclaim incomplete behavior or evaluation results.
